This project is written in Python 3 running in a Jupyter Notebook. Assuming previous installation of Anaconda, a user simply needs to run the Jupyter Notebook "526_a2.ipynb" by typing 'jupyter notebook' in the command line in the corresponding directory. Running the notebook will then run all of the cells of the notebook sequentially to be provided the seamless and non-seamless cloning results.

User provided alterations should be made in the second cell of the notebook.

  • Image files can be swapped. The user changes the file path names for the destination, mask, and source image variables: 'destination_filename,' 'mask_filename,' and 'source_filename.'
  • Source offset from the upper right corner of the image can be altered. If the images are perfectly alligned with respect the their upper left corners, the user should enter (0,0) here as offset.
  • Monochrome transfer of the source image can be toggled. To initiate monochrome transfer, the user should set the value of 'src_type' to 'MONOCHROME_SRC.'
